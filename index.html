WEB DEV
Front-end: Es el desarrollo web en el ámbito del cliente, es decir, en el navegador web.
Back-end: Es el desarrollo web en el ámbito del servidor, donde las tecnologías y lenguajes son directamente relacionadas con el sistema operativo del servidor de la página web.
